# -SLIMAPP-v2.0-Sistema-de-Gesti-n-Sindical-Modular-
SLIMAPP v2.0 es la evolución integral del ecosistema de gestión para el Sindicato SLIM N°3. Esta versión ha sido completamente refactorizada bajo una arquitectura modular, optimizando el rendimiento, la escalabilidad y el mantenimiento del código en entornos de Google Apps Script.
